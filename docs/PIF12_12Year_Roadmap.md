# PIF12 十二年 Roadmap（2026–2037）

**版本：** 1.0
**建立日期：** 2026-04-12
**維護者：** Jason J. Lai + Claude（AI 協作夥伴）

## 總覽

PIF12 是一個跨越 12 年（一個完整生肖輪迴）的 Web3 社會實驗。本文件定義五大管理領域在每一年的具體目標與里程碑，作為長期推進的主幹文件。

### 五大管理領域

- **(A) 程式碼** — 智能合約、前端、腳本
- **(B) 行銷與宣傳** — 網站、社群媒體、活動、品牌
- **(C) 圖檔與視覺資產** — Omamori 設計、IPFS metadata、品牌素材
- **(D) 上鏈與多簽管理** — 合約操作、Safe 多簽、IPFS、鏈上狀態
- **(E) 社群經營** — 成員關係、治理、線下活動、文化建設

## 風險與應變

| 風險 | 影響 | 應變計畫 |
|------|------|---------|
| Base 鏈停運/退場 | 合約不可用 | 合約可升級，可遷移至其他 EVM 鏈 |
| Jason 無法繼續主導 | 項目停滯 | Social Recovery 機制 + Safe 多簽保證社群可接手 |
| 核心成員流失 | 社群萎縮 | Karma + 年度聚會維持連結；允許新成員加入 |
| 合約漏洞被利用 | 資產/身份損失 | pause() 緊急凍結 + UUPS 升級修復 |
| IPFS 內容遺失 | Metadata 不可讀 | 多重 pin（Pinata + 本地備份 + web3.storage） |
| 私鑰遺失 | 無法操作合約 | Safe 多簽 + Social Recovery 雙重保險 |
